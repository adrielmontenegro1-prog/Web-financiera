COMO ACTIVAR GUARDADO ONLINE EN VERCEL + SUPABASE

IMPORTANTE
Los datos financieros se guardaran en tu proyecto privado de Supabase.
No pongas claves privadas dentro de index.html ni en GitHub.

1. Crea un proyecto en Supabase

Ve a:
https://supabase.com

Crea un proyecto nuevo.

2. Crea la tabla

En Supabase, abre SQL Editor.
Copia y ejecuta el contenido del archivo:

SUPABASE-SETUP.sql

3. Copia las variables

En Supabase ve a:
Project Settings > API

Necesitas:

SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY

La SERVICE_ROLE_KEY es privada.
NO la subas a GitHub.

4. Agrega variables en Vercel

En Vercel:
Project > Settings > Environment Variables

Agrega:

SUPABASE_URL = tu URL de Supabase
SUPABASE_SERVICE_ROLE_KEY = tu service role key

Guarda.

5. Redeploy

En Vercel:
Deployments > Redeploy

6. Prueba

Abre tu web.
Ve a Configuracion.
Debe decir:

Nube conectada
o
Guardado online

Si dice Local, Vercel no tiene las variables o la tabla no esta creada.

ARCHIVOS QUE DEBES SUBIR A GITHUB / VERCEL

index.html
vercel.json
api/load.js
api/save.js
SUPABASE-SETUP.sql
CONFIGURAR-GUARDADO-ONLINE.txt
